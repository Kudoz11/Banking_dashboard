# Banking Operations & Loan Analysis Dashboard 🏦

### Project Objective
This project involves the creation of a comprehensive Banking Dashboard designed to analyze a large dataset (100,000+ records). The goal is to monitor banking operations, visualize loan distributions, and provide real-time insights into financial health and customer segments.

### Tech Stack
* **Tool:** Power BI Desktop
* **Data Source:** SQL Server / Excel (Mention your source here)
* **Features:** DAX Queries, Power Query, Dynamic Filtering

### Key KPIs Analyzed
* **Total Sales/Revenue:** High-level view of bank's financial inflow.
* **Loan Status:** Distribution of current, paid, and overdue loans.
* **Customer Demographics:** Analysis by age, region, and job type.
* **Rating & Satisfaction:** Average customer feedback scores across different service sectors.
