# 🏦 Banking Operations & Loan Analysis Dashboard
### *Enterprise-level Data Visualization of 100K+ Financial Records*

---

## 📊 Project Overview
This project focuses on transforming raw banking data into actionable insights. Using **Power BI**, I analyzed a dataset of over 100,000 transactions to help bank managers monitor loan health and customer segments in real-time.

---

## 🛠️ Data Engineering & Tech Stack
- **Power BI:** Dashboard design and visualization.
- **DAX (Data Analysis Expressions):** Created complex measures for Rolling Averages, Year-over-Year (YoY) Growth, and Loan Default Rates.
- **Power Query:** Performed ETL (Extract, Transform, Load) to clean null values and standardize currency formats.
- **Python (Jupyter):** Initial data profiling and cleaning.

---

## 💡 Key Insights Delivered
1. **Risk Management:** Identified a 12% increase in overdue loans in the 'Senior' demographic segment.
2. **Regional Performance:** Found that 40% of the bank's revenue is generated from only 3 specific regions.
3. **Customer Profiling:** Segmented customers by job type to target high-value loan products.
